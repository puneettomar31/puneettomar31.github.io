🛠️ Tech Stack

Laravel Livewire MySQL Tailwind CSS REST API PHP 8.x Blade Templates Laravel Sanctum Vite

📋 Project Overview

The EdTech Course Management System is a comprehensive learning platform built to handle the full student lifecycle — from registration to course completion. This project is inspired by and closely mirrors the EdTech platform I build at SciAstra, where students prepare for competitive exams like IAT, NEST, JEE, and NEET.

The system provides course creators with tools to manage content, and students with an intuitive interface to enroll, track progress, and access learning material — all powered by a robust Laravel backend with Livewire-driven reactive UI.

✅ Key Features

  • Student registration & authentication with email verification
  • Course catalog with categories, filters, and search functionality
  • Student enrollment system with enrollment status tracking
  • Course module management — create, update, delete lessons and chapters
  • Progress tracking — students can mark lessons complete, see % progress
  • REST API secured with Laravel Sanctum for mobile/frontend integration
  • Livewire for real-time search, enrollment, and progress updates
  • Admin panel for managing students, courses, and enrollments
  • MySQL database with optimized queries and proper indexing
  • Built using GitHub Copilot & ChatGPT to streamline development

🏗️ Architecture & Design

  • MVC Architecture — clean separation of Controllers, Models, and Views
  • Service classes for complex enrollment and progress business logic
  • Eloquent relationships: User → Enrollments → Courses → Modules
  • Laravel Policies for student/admin access control
  • API Resources & Collections for consistent JSON API responses
  • Database migrations & seeders for reproducible environments
  • Tailwind CSS for mobile-first, responsive design
  • Livewire components reduce full page reloads for a smoother UX
